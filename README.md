# БИТВА ТРЕНЕРОВ · GYM — прототип Telegram Mini App

**Уровень: DEMO.** Балансы живут в памяти телефона (localStorage) — сервера нет.
Этого достаточно, чтобы открыть в Telegram, всё понажимать и показать людям.

## Что внутри

| Путь | Что это |
|---|---|
| `index.html` | Всё приложение одним файлом: Главная (тапалка-гиря), Споры 1×1 (карусель 5 споров), Задания/Очки действия (конвертер GYM→ACT, спонсоры, NFT, розыгрыш), Донаты (Синие/Красные), Голосования (3 события с таймерами), Прокачка, Друзья |
| `assets/` | Все картинки: фото людей, события, NFT, спонсоры, вырезанная гиря, QR-коды |

Экономика и контент (споры, имена, события, курсы) — в объекте `CONFIG`
в начале `<script>` в `index.html`. Правится текстом, без знания кода.

## Запуск за 5 минут

### Шаг 1 — залить в репозиторий
В репозитории `bitva-trenerov` на GitHub:
1. Удали старый `index.html` и архив с фото (они больше не нужны).
2. **Add file → Upload files** → перетащи `index.html` и папку `assets` целиком → **Commit changes**.

### Шаг 2 — включить GitHub Pages
1. В репозитории: **Settings → Pages**.
2. Source: **Deploy from a branch** → Branch: **main** / **(root)** → **Save**.
3. Через 1–2 минуты приложение живёт по адресу:
   `https://saleedariva-gif.github.io/bitva-trenerov/`
   Открой в браузере телефона — должно работать уже здесь.

### Шаг 3 — привязать к боту
1. В Telegram открой **@BotFather**.
2. Если бота нет: `/newbot` → имя → юзернейм.
3. `/mybots` → выбери бота → **Bot Settings → Menu Button → Configure Menu Button**.
4. Вставь URL из шага 2, название кнопки — например `В БОЙ 🥊`.
5. Открой чат с ботом → кнопка меню слева от поля ввода → приложение открывается внутри Telegram.

## Что нажимается (для демонстрации)
- Тап по гире → +5 GYM, счётчик и уровень растут, вибрация.
- Плитки Прокачка / Споры → подэкраны, стрелка ‹ — назад.
- Споры: свайп карусели, ставка списывает ACT, через ~3 сек случайный итог, счёт 43/16 обновляется.
- Задания: конвертер 7000 GYM → 15 ACT реально меняет балансы; розыгрыш стоит 50 ACT.
- Донаты: переключение Синие/Красные меняет неон, выбор участника, отправка списывает GYM.
- Голосования: таймеры идут в реальном времени, голос −150 ACT.
- Друзья: поиск фильтрует, «Пригласить друга» открывает шэринг Telegram.

## Сброс демо-состояния
Балансы сохраняются на телефоне. Чтобы начать «с нуля» перед показом:
в Telegram → настройки бота → «Очистить данные» у Mini App, либо открой
приложение и в консоли браузера выполни `localStorage.clear()`.
